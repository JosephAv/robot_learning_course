Course Description

This course will provide foundational knowledge on intelligent systems and their high-level control, aimed at developing robots with embodied intelligence able to learn and interact with their environment. Lectures focus on machine learning solutions for autonomous systems, especially Reinforcement and Imitation Learning techniques.

Course Modules

Robotics Survival Kit (approx. 10h)

  • Foundations of sensing and actuation
  • Kinematics and dynamics of autonomous systems
  • Motor control
  • Introduction to ROS programming (optional)
  • Gaussian Filters, Kalman Filter, Particle Filter (optional)

Intro to Reinforcement Learning

  • Supervised Learning for Robotics
  • Imitation Learning vs Reinforcement Learning
  • Markov Decision Processes

Value-based RL

  • Planning by Dynamic Programming
  • Model-Free Prediction
  • Model-Free Control
  • Value Function Approximation

Policy Gradients

  • Finite Difference Policy Gradient
  • Monte-Carlo Policy Gradient
  • Actor-Critic approaches
  • Natural gradient
  • Deep policy networks

Model-based RL

  • Model-based learning
  • Integrated Architectures
  • Learning and Planning
  • Real and simulated experience
  • Monte-Carlo Tree Search
  • Go use-case
